# Examples Gallery

All examples were generated by Claude using the bundled excalidraw skill. The skill defaults to **dagre-layout.js** for automatic positioning — only sequence diagrams use the CLI approach with explicit coordinates.

Every example produces three output files: `.excalidraw` (editable source), `.svg` (scalable embed), `.png` (raster preview).

To generate your own: `excalidraw-agent-cli install-skill`, then describe what you want.

## Approach A — Dagre layout (default)

Write a `graph.json` describing nodes, edges, and optional zones — no x/y coordinates. `dagre-layout.js` computes positions automatically.

```bash
DAGRE=$(python3 -c "import excalidraw_agent_cli,os; print(os.path.join(os.path.dirname(excalidraw_agent_cli.__file__),'..','dagre-layout.js'))")
node "$DAGRE" examples/<folder>/graph.json --output examples/<folder>/diagram.excalidraw
excalidraw-agent-cli --project examples/<folder>/diagram.excalidraw export png --output examples/<folder>/diagram.png --overwrite
```

**Use for**: everything — architecture, flowcharts, mind maps, ER, state, CI/CD, data pipelines.

### A14 — Freeform — Agentic Debug Session

📁 [`agent-debug-session/`](./agent-debug-session/)

**Prompt**:
```
Visualize a Claude Code session debugging a memory leak. No template — describe
the story: OOM alert at 3am → 3 parallel reads (error log, codebase grep, db/pool.js)
→ 3 competing hypotheses as diamond nodes (event listener leak? cache without TTL?
DB connection never released?) → investigate each in parallel → two dead-end paths
(gray, ❌ ruled out) and one successful path (green, ✅ found in pool.js:47) →
fix with conn.release() → load test → memory stable at 180 MB → postmortem.
Gray out dead-end nodes. Make the successful path bolder (width: 3).
```

**No template.** The graph structure encodes the story — shapes carry meaning (diamond=hypothesis, ellipse=observation), colors encode state (blue=gathering, yellow=uncertain, green=resolved, gray=ruled out). Shows the tool as a general-purpose reasoning visualizer, not just an architecture diagrammer.

## Reproducing dagre examples

```bash
DAGRE=$(python3 -c "import excalidraw_agent_cli,os; print(os.path.join(os.path.dirname(excalidraw_agent_cli.__file__),'..','dagre-layout.js'))")

node "$DAGRE" examples/<folder>/graph.json --output examples/<folder>/diagram.excalidraw

excalidraw-agent-cli --project examples/<folder>/diagram.excalidraw \
  export png --output examples/<folder>/diagram.png --overwrite

excalidraw-agent-cli --project examples/<folder>/diagram.excalidraw \
  export svg --output examples/<folder>/diagram.svg --overwrite
```
